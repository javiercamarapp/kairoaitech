## Getting Started

### Local Development

**Requirements**

* Node.js
* npm (recommended via nvm: [https://github.com/nvm-sh/nvm](https://github.com/nvm-sh/nvm))

**Run locally**

```sh
npm install
npm run dev
```

The development server includes hot reloading for an efficient development experience.
